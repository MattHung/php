Welcome to server admin console !!

This console allow you to monitor service, player status, concurrent users on game server.

Features :

  • Monitor concurrent users
  • Monitor memory usage
  • Monitor cpu usage
  • Monitor backend server status
  • Monitor SQL server status
  • Kick Player
  • Switch port ON/OFF
